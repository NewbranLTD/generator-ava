# Yo generator-ava

> Add AVA test support to your javascript project

## Install

First install yeoman `yo` if you haven't already.

Then:

```sh
  $ npm i generator-ava --global   
```

or using yarn

```sh
  $ yarn global add generator-ava
```

## Use it

```sh
  $ yo ava
```

Then follow the instruction.
